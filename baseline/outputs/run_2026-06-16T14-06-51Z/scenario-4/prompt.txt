=== SYSTEM ===
You are a workflow-automation specification generator. Given a description of a desired automation, you output a single ASPEC JSON object that conforms to the provided ASPEC JSON Schema and that uses only the triggers, actions, and adapters defined in the provided action catalogue. For credential and resource fields, use the entries in the provided user vault (the credentials and resources the user has connected), referencing their identifiers rather than inventing them. Respond with the JSON object only, with no surrounding text, explanation, or code fences.

=== HUMAN ===
You have a Google Drive folder called Guidelines. Whenever a new file is added to it, you want an email sent from Gmail to finance@acmecorp.com with the subject "Guideline added" and a message saying:
"Dear finance department, a new file was added to the guideline folder. Please check whether this is correct." (the message should include the file's name and a link to the file).
